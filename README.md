# Platform Game Using LCD And Adruino

Simple platform game using an LCD, button, and Arduino Mega 2560. When button is held, character will jump over obstacles.      


**Hardware:**    
-Arduino Mega 2560    
-16x2 LCD Screen    
-1 button    
-10k Potentiometer    
-Jumper wires     
   

**Wire Connection:**
|Arduino |  LCD  | Jump Button  | Potentiometer |
|--------|-------|--------------|---------------|
|5V      |A      |              |Front Left     |
|GND     |K      |Left          |Front Back     |
|Digital Pin 2|  |Right         |               |
|        |RS     |              |Back Leg       |
|Digital Pin 7|VSS|             |               |
|Digital Pin 8|E |              |               |
|Digital Pin 9|D4 |             |               |
|Digital Pin 10|D5|             |               | 
|Digital Pin 11|D6|             |               |
|Digital Pin 12|D7|             |               |
